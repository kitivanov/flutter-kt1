# KT1 - Dart OOP

Проект выполнен на языке Dart.

## Описание
Реализована модель с использованием объектно-ориентированного подхода.

## Возможности
- Использование конструктора
- Инкапсуляция
- Наследование
- Переопределение методов
- Статический метод

## Запуск
1. Убедитесь, что установлен Dart SDK
2. Откройте терминал в папке проекта
3. Выполните команду:

```
dart run main.dart
```
